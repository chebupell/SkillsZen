## 2. Описание моей работы

В рамках разработки **SkillsZen** я взял на себя ответственность за создание фундаментальных модулей приложения: системы безопасности (Auth), персонального пространства пользователя (Profile) и ключевых обучающих инструментов (JS Editor, AI Chat).

**Краткие итоги процесса:**
*   **Архитектура:** Участвовал в декомпозиции проекта на логические компоненты и ведении GitHub Dashboard.
*   **Технологии:** Использовал React в связке с Firebase для обеспечения надежного бэкенда и облачного хранения данных.
*   **Интеграция AI:** Внедрил GROQ LLM для создания интерактивного помощника, реализовав стриминг данных для лучшего UX.
*   **Качество:** Провел масштабную работу над тестированием, закрыв критические узлы (Auth, Logic) тестами, что позволило достичь 60% покрытия.

## 3. Личные Feature Components (Акцент для презентации)

Ниже описаны два компонента, разработанные мной полностью самостоятельно, которые я готов детально продемонстрировать на защите:

### 1. Система выполнения JS-кода (JS Coding Tasks)
Это мой самый сложный технический компонент. Он объединяет в себе:
*   Интегрированный редактор кода (**Code Editor**), где пользователь пишет решение.
*   Логику перехвата и исполнения кода в изолированной среде для сравнения результата с ожидаемыми тестами.
*   Динамическое обновление UI на основе успешности прохождения проверок.

### 2. Интеллектуальный AI-ассистент с поддержкой контекста
Компонент, демонстрирующий навыки работы с современными API:
*   **Стриминг ответов:** Реализован посимвольный вывод (streaming), что делает общение с ИИ "живым".
*   **Работа с данными:** Все сообщения автоматически привязываются к текущему пользователю в Firebase Firestore, обеспечивая сохранение контекста между сессиями.
*   **Native Integration:** Работа с GROQ API реализована через чистый `fetch` и `ReadableStream`, что минимизирует лишние зависимости в проекте.
